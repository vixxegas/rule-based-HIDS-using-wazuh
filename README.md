# rule-based-HIDS-using-wazuh

This project implements a rule-based Host Intrusion Detection System (HIDS) using Wazuh. It involves deploying a Wazuh Manager, a monitored Ubuntu client, and an attacker Kali VM to generate and detect malicious activity using custom detetection rules.

main tools and OS:
- Wazuh 4.14
- Ubuntu Server 22.04, Manager + Client
- Kali Linux, Attack VM
- VirtualBox
- Bash
